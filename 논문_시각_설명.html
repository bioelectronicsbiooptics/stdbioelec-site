PART 01

왜 DNA에 데이터를 저장하려 하는가?

전 세계 데이터는 2025년 175 제타바이트에 달합니다. 기존 저장장치로는 한계가 있어, 과학자들은 자연의 저장 매체인 DNA에 주목하고 있습니다.

💾

초고밀도 저장

DNA 1g에 455 엑사바이트 저장 가능.
전 세계 인터넷 데이터를 구두 상자 하나에 담을 수 있는 양입니다.

⏳

초장기 보존

적절한 조건에서 수천~수만 년 보존 가능.
하드디스크(5-30년)와는 차원이 다른 수명입니다.

🔋

무전력 유지

저장 유지에 전력이 필요 없습니다.
데이터센터의 막대한 냉각/전력 비용을 해결할 수 있습니다.

데이터 → DNA 변환 원리

컴퓨터의 이진법(0, 1)을 DNA의 네 가지 염기(A, T, G, C)로 변환합니다.

디지털 데이터 (이진법)

00011011 00110110
↓ 변환 규칙: 00=T, 01=A, 10=G, 11=C

DNA 염기 서열

TAGC TCAG

이렇게 사진, 영상, 문서 등 모든 디지털 파일을 DNA로 변환할 수 있습니다.

PART 02

현재의 문제점: 왜 아직 실용화되지 못했나?

DNA 저장은 2012년에 처음 시연되었지만, 핵심적인 기술적 병목 때문에 아직 실용화되지 못하고 있습니다.

문제 1: 읽기가 너무 느림

기존 PCR(DNA 복사 기술) 기반으로 데이터를 읽으려면 NGS 준비에만 6~12시간이 걸립니다. USB를 꽂고 파일을 여는 데 반나절이 걸리는 셈입니다.

문제 2: 수작업 의존

숙련된 기술자가 직접 시약을 준비하고, 프라이머를 선택하고, 라이브러리를 만들어야 합니다. 자동화가 어렵고 실수가 잦습니다.

문제 3: 일회용

대부분의 기존 방식은 DNA를 한 번 읽으면 더 이상 사용 불가. USB를 한번 읽고 버리는 것과 같습니다.

📚

비유로 이해하기

기존 DNA 저장은 "거대한 도서관에서 사서가 수작업으로 책을 찾아 복사해주는 것"과 같습니다. 찾고 싶은 책(데이터)의 고유 번호(프라이머)를 알려줘야 하고, 복사(PCR)에 시간이 오래 걸리며, 한번 꺼내면 원본이 손상됩니다.

PART 03

전기장 DNA 칩: 전기장으로 DNA를 직접 제어

반도체 칩 위의 마이크로전극에 전기장을 걸어 DNA를 선택적으로 읽는 완전히 새로운 방식을 제안합니다.

💡

핵심 비유

기존: "사서가 수작업으로 책을 찾아 복사" → 전기장 DNA 칩: "전자 도서관에서 버튼 하나로 원하는 책이 자동 출력"

마이크로전극 어레이 칩 (인터랙티브)

전극을 클릭하면 저장된 데이터가 나타납니다. 다시 클릭하면 사라집니다.

전극을 클릭해보세요!

작동 과정 4단계

1

DNA 고정

전극 표면에 DNA(데이터)를 화학적으로 고정. 전기장 이용시 20분이면 완료

2

프라이머 결합

전기장 ON인 전극에서만 프라이머가 결합. 같은 프라이머로 다른 전극의 다른 데이터 접근 가능

3

DNA 합성

Klenow 효소가 상온에서 DNA 복제. 전기장이 재료(dNTP)를 전극으로 모아줌

4

추출 & 분석

합성된 DNA 추출 후 20분 내 NGS 준비 완료. 디코딩으로 원본 복원

핵심 혁신: 공통 프라이머 (Common Primer)

기존 방식에서는 각 데이터마다 고유한 프라이머가 필요했습니다 (100개 데이터 = 100종 프라이머).
전기장 DNA 칩에서는 하나의 동일한 프라이머로 어떤 전극의 데이터든 접근할 수 있습니다.
전기장이 "어느 전극을 켤지"만 결정하면, 같은 프라이머가 해당 전극의 DNA에만 결합합니다.

마치 만능 열쇠(프라이머)는 하나인데, 전기 스위치(전기장)로 어느 방(전극)을 열지 선택하는 것과 같습니다.

PART 05

DNA Processing-in-Memory (메모리 내 처리)

저장된 DNA 데이터를 꺼내지 않고 그 자리에서 바로 조합/처리할 수 있습니다.

💻

컴퓨터로 비유하면

일반 컴퓨터: 하드디스크(저장) → RAM(메모리) → CPU(처리) 순서로 데이터를 옮겨야 합니다.
Processing-in-Memory: 저장소에서 바로 처리! 데이터 이동 없이 더 빠르고 효율적입니다.

예시: 문장 조합하기

전극 A

"We"
+

전극 B

"love"
+

전극 C

"DNA."
→

Assembly PCR

"We love DNA."

전극 B 대신 전극 D를 선택하면:

전극 A

"We"
+

전극 D

"synthesize"
+

전극 C

"DNA."
→

Assembly PCR

"We synthesize DNA."

Assembly PCR이란?

여러 DNA 조각을 순서대로 이어붙이는 기술입니다.
각 조각의 끝부분이 서로 겹치게 설계되어 있어서, 마치 레고 블록처럼 자동으로 맞물려 연결됩니다.
전기장 DNA 칩에서는 각 단어를 담은 DNA 조각에 "주소(토큰 프라이머)"가 달려 있어, 올바른 순서로 조립됩니다.

PART 06

대용량 데이터 저장 실험

0.2 MB 크기의 3D 문화유산 파일을 DNA에 저장하고 복원하는 데 성공했습니다.

인코딩

3D 파일 → DNA 변환

0.2 MB 3D 문화유산 데이터를 1,339개의 고유 DNA 가닥(각 150 뉴클레오타이드)으로 변환. Reed-Solomon 오류 정정 코드 + XOR 중복성 코드 추가.

어댑터 결합

NGS 어댑터 부착

DNA 가닥에 NGS 어댑터(시퀀싱 장비 인식용) 결합. 총 275 뉴클레오타이드 길이의 DNA 풀 생성.

칩 고정

전기장으로 전극에 고정

DNA 풀을 전기장 제어를 통해 마이크로전극에 고정. 전극 하나에 이론적으로 11억 개(1.1 x 109)의 분자 저장 가능.

합성 & 추출

전기장 기반 DNA 합성

전기장으로 프라이머 결합 → Klenow 효소로 DNA 복제 → 추출. 20분 PCR 프로토콜로 NGS 라이브러리 준비 완료.

디코딩

NGS 분석 & 데이터 복원

96.56% 완벽 일치율, 중간 깊이 10,337.9 reads. 합의 시퀀싱 + XOR 디코딩으로 3D 파일 완벽 복원.

왜 96.56%면 충분한가?

DNA 데이터 저장에는 오류 정정 코드(Reed-Solomon, XOR)가 포함되어 있습니다.
일부 DNA 가닥이 완벽하지 않더라도, 오류 정정 알고리즘이 자동으로 복구합니다.
마치 QR코드가 일부 손상되어도 스캔 가능한 것과 같은 원리입니다.
실제로 이 실험에서 3D 파일은 완벽하게 복원되었습니다.

PART 07

의의와 남은 과제

전기장 DNA 칩이 DNA 데이터 저장 분야에서 갖는 의미와, 앞으로 해결해야 할 과제들입니다.

전기장 DNA 칩의 3가지 핵심 의의

1. 분자 + 전자의 융합

DNA(분자)를 반도체 칩(전자)으로 직접 제어하는 새로운 패러다임. 화학/생물학적 방법에서 전자적 제어로의 전환.

2. 실용화에 한 걸음

20분 NGS 준비, 10만 번 재사용, 공통 프라이머 — DNA 저장의 주요 병목을 모두 해결하는 방향.

3. 반도체 공정 호환

기존 반도체 제조 공정으로 칩 제작 가능 → 전극 수를 늘려 용량 확장이 용이. 메모리 칩과 같은 확장 경로.

남은 과제 (리뷰어 지적 포함)

확장성 검증

현재 4개 전극 시연 → 수백~수천 개에서도 작동하는지 확인 필요. 전극 간 DNA 확산(crossover) 문제 검토.

최대 DNA 길이

합성 가능한 최대 가닥 길이와 효율 관계 규명 필요. 길이가 길어야 저장 밀도가 높아짐.

공정한 시간 비교

칩에 DNA를 처음 고정하는 시간도 포함하여 총 시간 비교가 필요하다는 리뷰어 지적.

용어 정리

"DNA 합성"이라는 표현이 새로운 데이터 쓰기로 오해될 수 있음. 실제로는 기존 데이터의 복제/읽기.

PART 08

핵심 용어 사전

논문에 등장하는 주요 용어들을 쉽게 풀어 설명합니다.

DNA (디옥시리보핵산)
생명체의 유전정보를 담는 분자. A, T, G, C 네 글자로 이루어진 매우 긴 코드. 이 연구에서는 디지털 데이터를 저장하는 매체로 사용됩니다.
뉴클레오타이드
DNA를 구성하는 기본 단위. A(아데닌), T(티민), G(구아닌), C(시토신) 각각이 해당. 디지털 비트(0,1)에 대응됩니다.
PCR (중합효소연쇄반응)
특정 DNA 조각을 수백만 배로 복사하는 기술. "DNA 복사기". 기존 DNA 저장에서 데이터를 읽는 핵심 수단이지만 느리고 오류가 발생합니다.
NGS (차세대 시퀀싱)
DNA 염기 서열을 대량으로 빠르게 읽는 기술. "DNA 판독기". 수백만 개의 DNA를 동시에 읽을 수 있지만, 시료 준비에 6-12시간이 걸립니다.
프라이머 (Primer)
DNA 복제의 시작점이 되는 짧은 DNA 조각(15~25글자). "책갈피" 역할. 전기장 DNA 칩의 핵심 혁신은 하나의 프라이머로 모든 전극에 접근하는 것.
Klenow Fragment
DNA 중합효소의 한 종류. 상온(실온)에서 DNA를 복제할 수 있는 특별한 효소. 높은 온도가 필요한 일반 PCR과 달리, 전극 위에서 안전하게 작동.
마이크로전극 어레이
아주 작은 금(Au) 전극들이 실리콘 칩 위에 격자 형태로 배열된 구조. 반도체 공정으로 제작. 각 전극에 독립적으로 전기장을 걸 수 있음.
어댑터 (Adapter)
NGS 장비가 DNA를 인식하고 읽을 수 있게 붙이는 특수 DNA 서열. 전기장 DNA 칩에서는 미리 어댑터를 포함하여 NGS 준비 시간을 20분으로 단축.
Random Access (랜덤 접근)
원하는 데이터에 순서 상관없이 바로 접근하는 능력. HDD/SSD처럼 아무 위치나 즉시 읽기. DNA 저장에서는 전극 선택으로 구현.
Reed-Solomon 코드
데이터에 오류가 있어도 자동으로 복구하는 수학적 방법. QR코드, CD, 위성 통신 등에도 사용. DNA 가닥 일부가 손상되어도 원본 복구 가능.
Processing-in-Memory
데이터를 저장소에서 CPU로 옮기지 않고, 저장소 안에서 바로 처리하는 방식. DNA 조각들을 직접 조합하여 새로운 정보를 생성.
Cross-electrode Contamination
인접한 전극의 DNA가 서로 섞이는 현상. 전기장 DNA 칩에서는 전극 간 오염이 거의 없음을 NGS로 확인하여 선택적 접근의 신뢰성을 입증.
