PART 01

왜 DNA에 데이터를 "인쇄"하려 하는가?

하드디스크는 10년이면 죽습니다. 문화유산 3D 데이터를 수천 년 보존하려면 완전히 새로운 매체가 필요합니다.

💿

현재 저장장치의 한계

HDD 5-10년, SSD 5-10년, CD 10-30년. 문화유산은 수백~수천 년 보존이 필요한데, 현재 기술로는 10-30년마다 복사해야 합니다.

🧬

DNA: 궁극의 저장 매체

1g에 455 엑사바이트 저장, 수천 년 보존 가능. 전력 불필요. 수만 년 된 화석에서도 DNA가 읽힌 사례가 있습니다.

⚠️

기존 DNA 저장의 3대 한계

1) 낮은 정보 밀도 (1-2 bit/nt)
2) 느린 디코딩 (수 분~수 시간)
3) 불완전한 정확도 (데이터 깨짐)

🖋️

DNA 잉크란?

프린터 잉크로 종이에 글자를 인쇄하듯, DNA 잉크로 금속판/유리에 디지털 데이터를 기록하는 것입니다. 잉크 안에는 수십만 개의 DNA 가닥이 들어있고, 각 가닥이 데이터의 일부를 담고 있습니다.

PART 02

DNA 잉크의 세 가지 핵심 혁신

밀도, 속도, 정확도 — 기존 DNA 저장의 세 가지 한계를 동시에 돌파합니다.

혁신 1: 3D DNA 번역층 (3D DTL) — 뉴클레오타이드당 9.78 bit

기존에는 텍스트를 그대로 DNA로 바꿔서 같은 글자가 반복되면 AAAA, TTTT 같은 호모폴리머(읽기 오류의 원흉)가 생겼습니다.
3D DTL은 3D 좌표 데이터를 숫자로 정규화한 뒤 DNA 4진법으로 직접 변환하여, 호모폴리머를 제거하고 밀도를 극대화합니다.

혁신 2: iSCAN — 기존 대비 360배 빠른 디코딩

기존: 수백만 개 시퀀싱 리드를 하나하나 전체 비교 (느림)
iSCAN: DNA 서열의 인덱스 위치만 스캔하여 즉시 분류 (빠름)

📚

비유

기존: 도서관에서 모든 책의 내용을 읽어서 비교 → iSCAN: 책등의 번호만 보고 바로 분류

혁신 3: 이중 오류 정정 — 완벽한 데이터 복원

Inner RS (Reed-Solomon): 각 DNA 가닥 내부에서 최대 32비트 오류 자동 복구 — "각 페이지의 자동 교정 기능"

Outer XOR: 가닥이 완전히 유실되어도 다른 가닥에서 복원 — "전체 백업 복사본"

결과: 데이터를 5.33배 줄여도(다운샘플링) 완벽한 복원 달성

PART 05

디코딩 파이프라인: 34.5초에 1.63GB

DNA에서 원본 데이터를 복원하는 전체 과정을 단계별로 살펴봅니다.

1

NGS 시퀀싱

DNA 풀을 차세대 시퀀싱으로 읽어 수백만 개의 raw 리드 생성

2

프라이머 분류

리드를 프라이머 서열 기준으로 분류하고 역상보 서열 생성

3

KIS 메타데이터

Key Information Sequence에서 파일 구조, 인덱스 정보 자동 추출

4

iSCAN 정렬

인덱스 스캔으로 서열을 초고속 분류 & 합의 시퀀싱으로 정확도 향상

5

RS 오류 정정

Inner Reed-Solomon 코드로 각 가닥의 오류 자동 복구

6

XOR 복원

유실된 가닥을 XOR 중복성으로 복원. Outer ECC 적용

7

3D DTL 역변환

DNA 서열 → 정수 → 3D 좌표로 역변환하여 원본 복원

8

데이터 출력

완벽한 3D 파일 복원 완료. 34.5초 (1.63 GB 기준)

PART 07

의의와 응용 가능성

DNA 잉크 기술이 여는 미래와 남은 과제입니다.

문화유산 영구 보존

직지, 석굴암, 숭례문 등 문화재의 3D 데이터를 DNA로 수천 년 보존. 물리적 훼손 시에도 디지털 복원 가능.

자율주행/AI 아카이빙

자율주행 3D 센서 데이터, AI 학습 데이터 등 대용량 데이터의 초장기 보존. 공간 효율적, 에너지 불필요.

자기 보존형 객체

문화재 표면에 DNA 잉크를 직접 도포 → 원본과 디지털 트윈이 하나의 객체에 공존.

VR/메타버스 콘텐츠

3D 공간 데이터를 DNA 잉크로 기록. 물리적 객체가 자신의 디지털 정보를 담고 있는 형태.

PART 08

핵심 용어 사전

논문에 등장하는 주요 용어들을 쉽게 풀어 설명합니다.

DNA 데이터 잉크
디지털 데이터를 인코딩한 DNA 가닥들의 용액. 물리적 표면에 도포하여 데이터를 저장할 수 있습니다.
3D DTL (DNA Translation Layer)
3D 좌표 데이터를 호모폴리머 없이 DNA로 고밀도 변환하는 번역 계층. 9.78 bit/nt 달성의 핵심.
iSCAN
Index Scan Cluster Alignment for Nucleotide. DNA 서열의 인덱스만 스캔하여 초고속 분류하는 디코딩 방법.
KIS (Key Information Sequence)
DNA 풀 안에 포함된 메타 정보 서열. 파일명, 크기, 인덱스 규칙 등을 담아 외부 정보 없이 디코딩 가능.
호모폴리머
AAAA, TTTT처럼 같은 염기가 연속 반복되는 서열. 시퀀싱 장비가 정확히 세지 못해 오류의 주원인.
Reed-Solomon 코드
데이터 오류를 자동으로 검출하고 복구하는 수학적 방법. CD, QR코드, 위성통신에도 사용됩니다.
XOR 중복성
여러 데이터 블록을 XOR 연산으로 결합하여 백업. 일부 가닥이 유실되어도 나머지에서 복원 가능.
올리고뉴클레오타이드
인공 합성된 짧은 DNA 가닥 (보통 150-300 nt). 데이터의 조각을 담는 기본 단위.
bit/nt
뉴클레오타이드(DNA 글자) 하나당 저장되는 비트 수. 높을수록 밀도가 높고 효율적.
다운샘플링
시퀀싱 데이터를 의도적으로 줄여서 최소한의 데이터로도 복원 가능한지 확인하는 실험.
Shannon 엔트로피
데이터가 얼마나 효율적으로 압축되었는지 측정하는 수학적 지표. 높을수록 정보 밀도가 높음.
직지심체요절 (Jikji)
1377년 고려에서 인쇄된 세계 최초 금속활자본. 이 연구에서 3D 스캔 데이터를 DNA로 보존한 대상.
