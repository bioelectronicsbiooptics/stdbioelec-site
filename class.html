Class track

LLM from Scratch

A hands-on class for building language models from the inside out: tokenization, attention, GPT architecture, pretraining, classification finetuning, and instruction finetuning.

Official LLM repository Reasoning follow-up

Course purpose

The class gives students enough implementation-level understanding to use LLMs responsibly in DNA data storage, bioinformatics automation, research writing, and lab software workflows.

Build the core

Implement token processing, attention, transformer blocks, GPT generation, training loops, and evaluation by hand.

Connect to research

Use LLMs as explainable tools for literature review, data annotation, manuscript drafting, and research automation.

Stay reproducible

Use official repositories for public links and keep local bilingual notes, HTML pages, and slides as teaching materials.

Class modules

The local teaching folder follows the main chapter structure of Build a Large Language Model (From Scratch), with Korean and English study materials.

Understanding LLMs

Model families, text generation, transformer overview, and where LLMs fit in scientific workflows.

Working with text data

Tokenization, vocabulary, dataloaders, batching, and how text becomes tensors.

Attention mechanisms

Self-attention, causal attention, multi-head attention, and the core computation behind transformers.

GPT model from scratch

Embeddings, transformer blocks, layer normalization, feed-forward layers, and text generation.

Pretraining

Training on unlabeled text, loss tracking, checkpointing, generation checks, and practical compute limits.

Finetuning

Classification finetuning, instruction finetuning, evaluation, and small-model adaptation for lab tasks.
