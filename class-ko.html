LLM from Scratch

토크나이저, attention, GPT 구조, 사전학습, 분류 finetuning, instruction finetuning까지 직접 구현하며 대형언어모델의 내부 원리를 이해하는 수업입니다.

HTML 전체 노트 열기 공식 LLM 저장소 Reasoning 후속 자료

수업 목적

학생들이 LLM을 단순히 사용하는 수준을 넘어, DNA 데이터 저장, 생물정보 자동화, 논문 작성, 연구실 소프트웨어 워크플로우에 책임 있게 적용할 수 있도록 구현 수준의 이해를 갖추게 합니다.

핵심 구조 직접 구현

텍스트 처리, attention, transformer block, GPT 생성, 학습 루프, 평가 과정을 직접 만들어 봅니다.

연구 업무와 연결

문헌 조사, 데이터 주석, 원고 초안, 연구 자동화에 LLM을 설명 가능한 도구로 쓰는 방식을 다룹니다.

재현 가능한 자료 관리

공개 링크는 공식 저장소를 사용하고, 한글/영문 노트와 HTML/PPT 자료는 로컬 강의 폴더에서 관리합니다.

수업 모듈

로컬 강의 폴더는 Build a Large Language Model (From Scratch)의 주요 챕터 구조를 따르며, 한글/영문 학습 자료가 함께 준비되어 있습니다.

LLM 이해

모델 계열, 텍스트 생성, transformer 개요, 과학 연구 워크플로우에서의 LLM 역할을 정리합니다.

텍스트 데이터 처리

토크나이징, vocabulary, dataloader, batching, 텍스트가 tensor가 되는 과정을 다룹니다.

Attention 메커니즘

Self-attention, causal attention, multi-head attention, transformer의 핵심 계산을 구현합니다.

GPT 모델 구현

Embedding, transformer block, layer normalization, feed-forward layer, text generation 흐름을 만듭니다.

사전학습

비지도 텍스트 학습, loss 추적, checkpoint, 생성 확인, 현실적인 계산 자원 한계를 함께 봅니다.

Finetuning

분류 finetuning, instruction finetuning, 평가, 연구실 업무에 맞춘 소형 모델 적응을 다룹니다.
